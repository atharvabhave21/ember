# /ember:follow-up

Draft a follow-up message to an existing contact using logged interaction history.

Usage: `/ember:follow-up [Name]`

Loads the 3 most recent interactions with the contact, recaps them for the user to confirm, then drafts one copy-paste ready message. Never re-introduces the user — always references something specific from the prior conversation.

Behaviour is defined in `skills/03-follow-up.md`.
