# /ember:setup

Run Ember's first-time onboarding flow.

Silently searches the user's Notion workspace for the Contacts database, Interactions database, and Ember Settings page. Based on what is found, either walks the user through setup or unlocks the full plugin.

Behaviour is defined in `skills/00-onboarding.md`.
